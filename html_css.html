HTML/CSS

Hypertext Markup Language (HTML) is a standard markup language. Cascading Style Sheets (CSS) is a style sheet language. HTML is for the structure of web pages and web applications, while CSS is for presentation.
I have learned to create web pages, how to upload them to a web server and access them with a browser. Also, create a CSS style sheet linked to HTML. Moreover, I learned to build responsive web pages and how to test them on different device resolutions. Personally, I am enjoying the class, specially the CSS part, because I like trying different styles and choosing the most appropiate. Two very helpful tools the teacher introduced to us are the HTML and CSS validators, both help me to script better and fix errors.
